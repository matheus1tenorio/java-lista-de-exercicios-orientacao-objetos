# Exercícios de Java

Repositório contendo exercícios realizados por mim durante a disciplina de Programação Orientada a Objetos utilizando Java.

Os exercícios tiveram como objetivo aprender e praticar os conceitos da linguagem Java e da orientação a objetos.

Os exercícios estão dentro da pasta `src`, separados por tópicos estudados.

## Conteúdo

As listas abordam temas como:

- Conceitos básicos da linguagem
- Estruturas de decisão
- Estruturas de repetição
- Arrays
- Manipulação de Strings
- Classes, atributos e métodos
- Abstração e encapsulamento
- Herança e polimorfismo
- Interfaces
- Tratamento de exceções
